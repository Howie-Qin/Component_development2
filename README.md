
# 组件式开发Web app 全栈

  项目主要包含:<br/>
  1.  fullPage实现单页面全屏滚动<br/>
  2.  canvas 画图实现相应饼图Pie，环图Ring等<br/>
  3. 各种柱状图，折线图的画法<br/>
  4.组件式开发思想以及编码规范的学习<br/>
